Ce depôt contient un mini-projet pomodoro
Explication de fonctionnement:

application pour gerer le temps en méthode pomodoro

Travail (25 min)
   ↓
Pause courte (5 min)
   ↓
Travail (25 min)
   ↓
Pause courte (5 min)
   ↓
Travail (25 min)
   ↓
Pause courte (5 min)
   ↓
Travail (25 min)
   ↓
Pause longue (15 min)
   ↓
fin de travail Aujourd'hui

si tu veux le tester plus vite diminue le temps sur les 3 setInterval (1000-> 10)

commande que tu doit faire 
npm install
npm run dev
